# PythonGame
Python Game for College Project based on  DaFluffyPotato Tutorial


Current Objectives are :
- Tilemap 
- Save and load Maps (done with json)
- Mapeditor
- Camera Handling
- Animation
- Player with Dash Attack, jump and jump from wall
